# Changelog

All notable changes to the Code Practices Plugin will be documented in this file.

## [1.0.0] - 2025-01-27

### Added
- Initial release with 13 coding best practices skills

#### Development Workflow Skills
- **explore-plan-code**: Read files → plan → approve → implement workflow
- **goal-driven**: Define success criteria for declarative task specification
- **test-first**: TDD approach with subagent verification
- **naive-then-optimize**: Correctness-first development pattern

#### Code Quality Skills
- **code-review**: Comprehensive review checklist with multi-Claude pattern
- **simplify**: Remove over-engineering and unnecessary abstractions
- **cleanup**: Systematic dead code and tech debt removal
- **surface-assumptions**: Identify and validate hidden assumptions
- **tradeoffs**: Present multiple approaches with explicit tradeoffs

#### Advanced Workflow Skills
- **visual-iteration**: UI feedback loop using screenshots
- **context-management**: Session management with /clear, checklists, subagents
- **multi-claude**: Parallel Claude instances for writer+reviewer patterns
- **headless-automation**: CI/CD integration and batch processing patterns
